# CRR Slotting & Equity

Quick-reference for specialised lending slotting risk weights and equity treatments.

**Regulatory Reference:** CRR Articles 147(8), 153(5), 155

---

## Specialised Lending Types

| Type | Abbreviation | Description |
|------|-------------|-------------|
| Project Finance | PF | Long-term infrastructure/industrial projects |
| Object Finance | OF | Ships, aircraft, physical assets |
| Commodities Finance | CF | Commodity inventory financing |
| Income-Producing RE | IPRE | CRE where repayment depends on rental income |
| High Volatility CRE | HVCRE | CRE with higher risk characteristics |

## CRR Slotting Risk Weights (Art. 153(5))

### Non-HVCRE (Table 1)

| Category | Maturity >= 2.5yr | Maturity < 2.5yr |
|----------|-------------------|------------------|
| Strong | 70% | 50% |
| Good | 90% | 70% |
| Satisfactory | 115% | 115% |
| Weak | 250% | 250% |
| Default | 0% | 0% |

### HVCRE (Table 2)

| Category | Maturity >= 2.5yr | Maturity < 2.5yr |
|----------|-------------------|------------------|
| Strong | 95% | 70% |
| Good | 120% | 95% |
| Satisfactory | 140% | 140% |
| Weak | 250% | 250% |
| Default | 0% | 0% |

## Equity Treatments

### SA (Art. 133)

| Equity Type | Risk Weight |
|-------------|-------------|
| Central bank / sovereign | 0% |
| Listed / exchange-traded | 100% |
| Government-supported | 100% |

### IRB Simple (Art. 155)

| Equity Type | Risk Weight |
|-------------|-------------|
| Exchange-traded / listed | 290% |
| Private equity (diversified) | 190% |
| All other (unlisted, speculative) | 370% |

**Note:** Basel 3.1 removes equity IRB — all equity falls to SA at 250%/400%.
